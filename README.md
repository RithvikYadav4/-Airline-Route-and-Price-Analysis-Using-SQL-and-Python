# -Airline-Route-and-Price-Analysis-Using-SQL-and-Python


# Overview
 1.Analyzed 300K+ flight records using Python (Pandas, Seaborn) and SQLite to uncover airline pricing
 behaviors and route-level trends.
 
 2.Performed extensive data cleaning, transformation, and exploratory analysis to compare fares across
 airlines, routes, travel classes, and booking windows.
 
 3.Used SQLite queries and Python visualizations to validate data, generate insights, and identify statistically
 significant pricing patterns.
 
 4.Found that Vistara had the highest average ticket price (~₹30,396) while AirAsia had the lowest (~₹4,091).
 
 5.Identified Delhi–Mumbai as the busiest flight route with approximately 59,000 recorded flights.
 
 6.Discovered that Delhi–Kolkata and Delhi–Chennai were the most expensive routes, with average fares
 around ₹21,900.

 7.Analyzed class-wise pricing and observed that Business Class fares varied significantly, with fluctuations of
 around ±₹12,969.
 Detected a clear trend showing that airline ticket prices rise sharply as the departure date approaches.
